# Accordion — Accessibility Requirements

Guide for implementing and testing accessibility of the Dreamy UI `Accordion` component.

**APG pattern:** [Accordion](https://www.w3.org/WAI/ARIA/apg/patterns/accordion/)

---

## 1. Purpose and semantics

An accordion is a stack of **collapsible sections**: each header contains a button that shows/hides a panel. Panels use `role="region"` labelled by their header button.

---

## 2. Component anatomy

| Part | Purpose in a11y |
| --- | --- |
| **`Accordion.Root`** | Accordion container / descendants |
| **`Accordion.Item`** | One header + panel pair |
| **`Accordion.Trigger`** | `<button>` inside a heading (`h2` default via `headingTag`) |
| **`Accordion.Content`** | Panel (`role="region"`, `aria-labelledby` → trigger) |
| **Icon** | Decorative chevron (`aria-hidden`) |

---

## 3. Requirements

### Trigger

- Native `<button type="button">` with accessible name (visible children).
- Wrapped in a heading element (`headingTag`, default `h2`) for document structure.
- `aria-expanded` reflects open state; `aria-controls` → panel id.
- Disabled items: not expandable; announced as disabled.

### Panel (`Accordion.Content`)

- `role="region"`, `id` referenced by `aria-controls`.
- `aria-labelledby` → trigger id.
- Hidden when collapsed (`hidden` / equivalent); content not reachable when closed.
- Optional height animation must respect `reduceMotion`.

### Keyboard (APG)

| Key | Requirement |
| --- | --- |
| **Enter** / **Space** | Toggle the focused header button |
| **Tab** | Move between header buttons (and into focusable content in an open panel) |
| **ArrowDown** / **ArrowUp** | Optional APG: move focus between headers |
| **Home** / **End** | Optional APG: first / last header |

---

## 5. Checklist

- [x] Trigger is a named `<button>` inside a heading
- [x] `aria-expanded` / `aria-controls` wired to the panel
- [x] Panel is `role="region"` with `aria-labelledby`
- [x] Collapsed panel content not exposed / focusable
- [x] Enter/Space toggles; focus visible
- [x] Icon is `aria-hidden`; reduced motion respected
