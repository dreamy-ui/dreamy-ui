# Textarea ‚Äî Accessibility Requirements

Multi-line text field. Dreamy uses a native `<textarea>` (`Textarea` with autosize, or `TextareaNoAutoSize`).

**APG:** Prefer native HTML textarea; see [APG Patterns](https://www.w3.org/WAI/ARIA/apg/patterns/). No custom role on a native textarea.

---

## Dreamy form composition

Standalone control; compose **inside `Field`**. Field owns label/hint/error. Textarea wires `id`, `aria-describedby`, `aria-invalid`, required/disabled via `useField`.

```tsx
<Field>
  <Field.Label>Bio</Field.Label>
  <Textarea />
  <Field.Hint>Max 500 characters.</Field.Hint>
  <Field.Error>Bio is required.</Field.Error>
</Field>
```

Placeholder is not a label.

## Anatomy

| Part | Element | A11y role |
| --- | --- | --- |
| **Textarea** | native `<textarea>` (autosize wrapper or plain) | Multiline textbox ‚Äî focus + value |

Autosize is visual/layout only; it must not break native semantics or keyboard entry.

---

## Requirements

### Role & value

- Native `<textarea>` ‚Äî implicit multiline textbox.
- Value exposed via the value property.
- Resize/autosize must not remove keyboard access or clip focus rings.

### Naming & Field wiring

| Attribute | Requirement |
| --- | --- |
| Accessible name | `Field.Label` or `aria-label` / `aria-labelledby` |
| `aria-describedby` | Field hint + error (+ character count if provided elsewhere) |
| `aria-invalid` | Field `isInvalid` |
| `required` | Field `isRequired` |
| `disabled` / `readOnly` | From Field / props |
| `id` | Shared with Field.Label |

### Keyboard

| Key | Behavior |
| --- | --- |
| **Tab** | Focus / leave the field |
| **Enter** | Insert newline (do not submit surrounding form unless user expects that) |
| Arrows / Home / End | Native caret movement |

No keyboard trap. If a character counter exists, expose it via `aria-describedby` or a live region only when updates are essential (prefer static description + visible count).

## Checklist

- [ ] Native textarea is the focusable control
- [ ] Accessible name from Field (or aria-label)
- [ ] Hint/error linked with `aria-describedby`
- [ ] Invalid / required / disabled states correct
- [ ] Enter inserts newlines; no unexpected trap
- [ ] Autosize does not hide content from AT or clip focus
- [ ] Visible focus indicator
- [ ] Placeholder not the sole label
