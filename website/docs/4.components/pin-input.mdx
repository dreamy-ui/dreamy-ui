## Import

- `PinInput` - The Pin Input component with context.
- `PinInputField` - The field for the pin input.

```tsx
import { PinInput, PinInputField } from "@dreamy-ui/react";
```

## Usage

`PinInput` does not have any wrapper by default, so you probably want to use `stacked` prop to tell Dream to wrap it into a `HStack`. Use `wrapperProps` to customize the `HStack` component.

<Wrapper>
	<PinInput stacked>
		<PinInputField />
		<PinInputField />
		<PinInputField />
	</PinInput>
</Wrapper>

```tsx
<PinInput stacked>
  <PinInputField />
  <PinInputField />
  <PinInputField />
</PinInput>
```

### Controlled

<Wrapper>
  <ControlledPinInput />
</Wrapper>

```tsx
export function ControlledPinInput() {
    const [pin, setPin] = useState("69");

    return (
        <PinInput
            value={pin}
            onChange={setPin}
            stacked
        >
            <PinInputField />
            <PinInputField />
            <PinInputField />
        </PinInput>
    );
}
```

### Invalid
Pass `isInvalid` prop to the `PinInput` to show the error state.

<Wrapper>
	<PinInput stacked isInvalid>
		<PinInputField />
		<PinInputField />
	</PinInput>
</Wrapper>

```tsx
<PinInput stacked isInvalid>
	<PinInputField />
	<PinInputField />
 </PinInput>
```
