### Max, Min, Step

You can change the max, min, and step of the Slider by using the `max`, `min`, and `step` props.

<Wrapper>
	<MaxMinSlider />
</Wrapper>

```tsx
function MaxMinSlider() {
    const [value, setValue] = useState(0);

    return (
        <>
            <Text>
                Slider value: {value}
            </Text>

            <Slider.Root
                min={0}
                max={50}
                step={10}
                value={value}
                onChangeValue={setValue}
                >
                <Slider.Track maxW="xs">
                    <Slider.FilledTrack />
                    <Slider.Thumb />
                </Slider.Track>
            </Slider.Root>
        </>
    );
}
```

### Controlled

You can control the Slider by using the `value` prop.
ControlledSlider

<Wrapper>
	<ControlledSlider />
</Wrapper>

```tsx
function ControlledSlider() {
    const [value, setValue] = useState(0);

    return (
        <>
            <Text>
                Slider value: {value}
            </Text>

            <Slider.Root value={value} onChangeValue={setValue}>
                <Slider.Track maxW="xs">
                    <Slider.FilledTrack />
                    <Slider.Thumb />
                </Slider.Track>
            </Slider.Root>
        </>
    );
}
```
