## Import

- `DarkTheme` - A simple Box with locked dark theme.
- `LightTheme` - A simple Box with locked light theme.

```jsx
import { DarkTheme, LightTheme } from "@dreamy-ui/react/rsc";
```

## Usage

<Wrapper>
	<DarkTheme>
		<Box p={2} rounded={"l1"} bg='bg'>This is a Theme</Box>
	</DarkTheme>
	<LightTheme>
		<Box p={2} rounded={"l1"} bg='bg'>This is a Theme</Box>
	</LightTheme>
</Wrapper>

```tsx
<DarkTheme>
	<Box p={2} rounded={"l1"} bg='bg'>This is a Theme</Box>
</DarkTheme>
```
