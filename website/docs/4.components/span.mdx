## Installation

```bash
pnpm dreamy add span
```

## Usage

Span is cool for styling a piece of text inside a paragraph.

<Wrapper>
	<Text>
		This is a paragraph with <Span fontWeight="bold">bold text</Span> using Span.
	</Text>
</Wrapper>

```jsx
<Text>
	This is a paragraph with <Span fontWeight="bold">bold text</Span> using Span.
</Text>
```
