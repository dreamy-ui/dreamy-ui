## Installation

```bash
pnpm dreamy add snippet
```

## Usage

<Wrapper>
	<Snippet w='full'>
		<>pnpm dlx dreamy add --all</>
	</Snippet>
</Wrapper>

```tsx
<Snippet>
	pnpm dlx dreamy add --all
</Snippet>
```

### Disable Tooltip

You can disable the "Copy To Clipboard" tooltip by using the `disableTooltip` prop.

<Wrapper>
	<Snippet disableTooltip w='full'>
		<>pnpm dlx dreamy add --all</>
	</Snippet>
</Wrapper>

```tsx
<Snippet disableTooltip>
	pnpm dlx dreamy add --all
</Snippet>
```
